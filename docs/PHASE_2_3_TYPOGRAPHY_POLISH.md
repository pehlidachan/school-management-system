# Phase 2.3 Typography Polish

This phase improves dashboard typography while keeping the dark purple/yellow theme.

## Changes

- Switched dashboard font to `Manrope`.
- Normal UI text is now regular weight.
- Sidebar menu items are regular weight.
- Buttons and table body text are regular weight.
- Headings remain bold.
- Table headers and active sidebar item are medium weight.
- Metric numbers remain bold for readability.

## Files changed

```text
school/templates/dashboard_base.html
static/css/typography.css
```

## Browser note

After pulling this update, use `Ctrl + F5` if the old font or old bold styling is cached.
