# Phase 2.2 Dark Purple + Yellow Dashboard Theme

This phase updates the dashboard CSS to match the supplied Usonacht-style reference image.

## Theme changes

- Dark purple sidebar.
- White sidebar text.
- Yellow sidebar heading labels.
- Yellow logo/accent buttons.
- Purple active menu item with yellow left indicator.
- Purple hero cards.
- Yellow/purple metric cards.
- White rounded content cards.
- Purple/yellow button theme.

## Main file changed

```text
static/css/dashboard.css
```

## Notes

This is a CSS-only theme update. Backend/database logic is unchanged.

After pulling this update, press `Ctrl + F5` in the browser if old colors are cached.
